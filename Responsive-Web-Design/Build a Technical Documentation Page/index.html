Introduction

Python is a high-level, interpreted programming language known for its readability and simplicity.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python is dynamically typed and garbage-collected.

It has a vast standard library and a large ecosystem of third-party packages.

It is commonly used in data science, web development, scripting, automation, and more.

print("Hello, World!")
Variables and Data Types

Variables in Python are created when you assign a value to them.

Data types include int, float, str, bool, list, tuple, dict, and set.

Python is dynamically typed, so you don't need to declare the type.

Use the `type()` function to check a variable's data type.

You can use `input()` to take user input.

x = 10 name = "Alice" price = 19.99
  • Integers
  • Floating-point numbers
  • Strings
  • Booleans
  • Collections: list, dict, set, tuple
Control Structures

Control structures allow you to control the flow of your program.

Common control structures include if, elif, else, for, and while.

Python uses indentation (typically 4 spaces) to define blocks.

Looping structures include `for` loops and `while` loops.

Use `break` and `continue` for loop control.

if x > 5: for i in range(5):
Functions

Functions in Python are defined using the `def` keyword.

They allow code reuse and modularity.

You can pass arguments and return values.

Python supports default parameters and keyword arguments.

Anonymous functions can be created using the `lambda` keyword.

def greet(name): return "Hello " + name

Object Oriented Programming

Python is an object-oriented language.

It supports classes and objects, inheritance, and encapsulation.

You define classes using the `class` keyword.

Each class can have methods (functions) and attributes (variables).

The `__init__` method is used as a constructor.

class Car: def __init__(self, model):
